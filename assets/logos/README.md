# Logos

`logo.png` 为西南财经大学校徽，已用于每个章节扉页右上角。

如需更换为自己的校徽，请用官方授权的文件替换 `logo.png`（建议保持透明背景），并在 `beamerthemeSWUFE.sty` 的 `section page` 模板中按需调整 `\includegraphics` 的尺寸或位置。

校名、校徽与视觉识别资源的使用需符合学校官方规定，使用者自行承担合规责任。

---

`logo.png` is the SWUFE emblem, used in the top-right corner of every section divider page.

To use a different logo, replace `logo.png` with an officially authorized file (a transparent background is recommended) and adjust the `\includegraphics` size/position in the `section page` template in `beamerthemeSWUFE.sty`.

University names, logos, and visual identity assets must be used lawfully and in accordance with official rules. The user is responsible for compliant use.
